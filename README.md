## ğŸš€ Getting Started

### Viewing the Website
Simply visit [https://youssefnkiza1.github.io/Web/](https://youssefnkiza1.github.io/Web/)

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/youssefnkiza1/Web.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Web
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js with http-server
   npx http-server
   ```

## ğŸ¯ Customization Guide

### 1. Personal Information
Update the following in `index.html`:
- Name and title
- Contact information
- Social media links
- About me text
- Skills list
- Project descriptions

### 2. Images
Replace placeholder images in `/images/`:
- `profile.jpg` - Your professional photo
- Project screenshots in `/images/projects/`
- Any other relevant images

### 3. Styling
Modify CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --text-color: #your-color;
    --background-color: #your-color;
}
```

### 4. Projects Portfolio
Edit the projects section in `index.html`:
- Add/remove project cards
- Update project titles, descriptions, and links
- Replace project images

### 5. Contact Form
Configure the contact form:
- Update form fields as needed
- Connect to a form service (Formspree, Netlify Forms, etc.)
- Or implement backend processing
